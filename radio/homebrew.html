Homebrew Receiver Specifications
Frequency Range 10 kHz to 55 MHz in 1-Hz increments
Frequency Stability Less than ± 1 x 10-9 at 0 to +40 ° C. Drift rate of 1 x 10-10 per month maximum.
First IF 70 MHz
Second IF 455 kHz
Sensitivity (SSB) 10 kHz to 500 kHz: 0.5 microvolt into 50 ohms for 10 dB S+N/N
500 kHz to 55 MHz: 0.25 microvolt into 50 ohms for 10 dB S+N/N
Noise figure 6.5 dB maximum
Third order intercept +30 dBm minimum at 20 kHz spacing
Intermodulation-free Dynamic Range 100 dB minimum
IF rejection 90 dB minimum
Image rejection 90 dB minimum
Internal spurious responses Less than 0.5-uV equivalent input
External spurious rejection 80 dB minimum at 20 kHz from tuned frequency
Audio output 5 Watts peak into 8 ohms, +10 dBm into 600 ohms; maximum 1% total harmonic distortion, 20 to 20,000 Hz
AGC control Less than 5 dB rise in audio output for 100 dB signal increase from noise floor.
IF Bandwidths 15.00 kHz first IF roofing filter, minimal group delay and phase delay characteristics
0.25, 0.6, 1.0. 1.8, 2.1, 2.5, 3.0, 4.5, 5.0, 6.5, 8.0, 10.0, 12.0 kHz bandpass filters
6/60 dB shape factor: 2.0 maximum.
Ultimate rejection 100 dB within first IF passband

Design Goals

  • Low noise
  • Extreme dynamic range
  • Ultra-linear circuitry
  • Broadband operation (i.e. no preselector tuning)
  • Computer controllable
  • Modular design
